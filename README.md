Лабораторна робота №6

ІО-35 Шевчук А.С. ФІОТ

Тема роботи

Налагодження модулів ядра Linux: виявлення та аналіз помилок виконання (kernel OOPS), створення інтерфейсів /proc, використання механізму seq_file для виведення діагностичної інформації та застосування інструментів налагодження ядра.

Мета роботи

Ознайомитися з методами налагодження модулів ядра Linux, навчитися викликати та аналізувати помилки ядра (kernel OOPS), створювати діагностичні інтерфейси у файловій системі /proc, працювати з API seq_file, застосовувати printk() і системні механізми збирання та перегляду налагоджувальної інформації.
